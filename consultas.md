## Ej. 3.2

### 13

```sparql
SELECT (COUNT(?p) as ?countp)
WHERE { ?p a foaf:Person }
```

### 14

```sparql
SELECT (COUNT(DISTINCT ?p2) as ?countp)
WHERE { 
  ?p1 a foaf:Person .
  ?p2 a foaf:Person .
  ?p1 (foaf:knows) ?p2 .
}
```

### 15

```sparql
SELECT (COUNT(DISTINCT ?p1) as ?countp)
WHERE { 
  ?p1 a foaf:Person .
  FILTER NOT EXISTS {
    ?p2 a foaf:Person .
    ?p2 (foaf:knows) ?p1 .
  }
}
```

### 16

```sparql
SELECT AVG(?ni) as ?avg
WHERE {
  ?p a foaf:Person.
  {
    SELECT (COUNT(?i) as ?ni)
    WHERE { ?p foaf:interest ?i }
    GROUP BY (?p)
  }
}
```
